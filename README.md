# grow-wedding
Wedding template built on growsdk and appengine.

## Getting started
- Check out the Grow Docs.
  - https://grow.io/docs/
- Install npm
  - https://nodejs.org
- Install packages found in package.json
```
  $ npm install
```
- Install bower components
  - http://bower.io/
```
  $ npm install -g bower
  $ bower install
```
